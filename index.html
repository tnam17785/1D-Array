Khái Niệm Trực Quan Cú Pháp Code Thao Tác Cơ Bản Bài Tập Kinh Điển

1. Mảng 1 Chiều Là Gì?

Mảng 1 chiều là một tập hợp các phần tử có cùng kiểu dữ liệu, được lưu trữ liên tiếp nhau trong bộ nhớ máy tính (RAM).

  • Chỉ số (Index): Bắt đầu từ 0 đến n-1 (với n là số phần tử).
  • Kích thước cố định: Khi đã khai báo, kích thước mảng thường không thể thay đổi (đối với mảng tĩnh).
  • Truy cập ngẫu nhiên: Có thể truy cập trực tiếp bất kỳ phần tử nào thông qua chỉ số với tốc độ cực nhanh O(1).

3. Cú Pháp Khai Báo & Khởi Tạo

Chọn ngôn ngữ bạn muốn xem:

// Khai báo mảng tĩnh 5 phần tử chưa có giá trị
int arr[5]; 

// Khai báo và khởi tạo giá trị ban đầu
int numbers[] = {10, 25, 8, 44, 12};

// Truy cập và thay đổi giá trị
numbers[1] = 30; // Thay 25 bằng 30
// Khai báo và cấp phát bộ nhớ cho mảng 5 phần tử
int[] arr = new int[5];

// Khai báo và khởi tạo nhanh
int[] numbers = {10, 25, 8, 44, 12};

// Gán giá trị mới
numbers[1] = 30;
# Python không có mảng tĩnh gốc, thường dùng List (Mảng động)
numbers = [10, 25, 8, 44, 12]

# Truy cập và thay đổi giá trị
numbers[1] = 30

# Python hỗ trợ duyệt ngược bằng chỉ số âm
print(numbers[-1]) # Lấy phần tử cuối cùng (12)

4. Các Thao Tác Cơ Bản Trên Mảng

Thao tác Giải thích Độ phức tạp (Độ nhanh)
Truy cập (Access) Đọc hoặc ghi giá trị tại một vị trí i biết trước. O(1) - Tức thời
Tìm kiếm (Search) Quét từ đầu đến cuối mảng để tìm một giá trị cụ thể. O(n) - Tuyến tính
Thêm / Xóa Chèn hoặc xóa phần tử ở giữa mảng (phải dịch chuyển các phần tử khác). O(n) - Chậm
